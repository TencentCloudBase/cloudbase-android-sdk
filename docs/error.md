# 错误码

👇 下面是常见的错误码及错误码说明

|           错误码            |            说明            |
| :-------------------------: | :------------------------: |
|        INVALID_PARAM        |       参数不符合预期       |
|         EMPTY_PARAM         |          参数为空          |
|         NETWORK_ERR         |          网络错误          |
|          JSON_ERR           |  JSONObject 数据解析失败   |
|           IO_ERR            |          IO 错误           |
|         AUTH_FAILED         |        获取授权失败        |
|    REFRESH_TOKEN_EXPIRED    |  身份认证过期，需重新登录  |
|     CHECK_LOGIN_FAILED      | 请求鉴权失败，身份信息异常 |
|          NOT_LOGIN          | 未登录，不存在身份认证信息 |
|          NOT_LOGIN          | 未登录，不存在身份认证信息 |
| INVALID_CUSTOM_LOGIN_TICKET |    非法的自定义登录票据    |
